# Task-007: 右ペイン（傍線部一覧 / 設問編集）

## フェーズ
Phase 03 - UI実装

## 概要
仕様書§5.1 の右ペインを実装する。傍線部のリスト表示、設問入力、順序変更、削除を行う。

## 作業内容

### 1. 傍線部リスト (`src/components/MarkList.tsx`)
- marks配列を順序通り表示
- 各アイテム:
  - 番号（①②③…）
  - 抜粋テキスト（excerpt）
  - 設問テキスト入力欄（textarea）
  - 順序移動ボタン（↑↓）
  - 削除ボタン

### 2. 設問入力
- 各傍線部に対応するtextarea
- 入力内容は `updateMarkQuestion` で state に反映
- 未入力時プレースホルダ: 「傍線部①について説明せよ。」（§6.5）

### 3. 順序変更
- ↑↓ボタンで `reorderMark` をdispatch
- 順序変更で番号が自動再採番される
- SVGオーバーレイも再描画

### 4. 削除
- 個別削除ボタン
- 全削除ボタン（confirmダイアログ必須）

### 5. 紙面の設問エリア
- `#paper` 内の `.question-area` に設問を表示
- 見出し: 問一、問二、… （傍線順に対応）
- 本文: ユーザ入力 or プレースホルダ

## 依存
- task-004（State管理）
- task-005（紙面DOM / 右ペイン枠）
- task-006（SVGオーバーレイ再描画）

## 完了条件
- 傍線部リストが右ペインに表示される
- 設問入力が紙面の設問エリアに反映される
- 順序変更で番号・設問が連動する
- 削除が正しく動作し、全削除にconfirmがある
